# üéì Course Management System (Console App)

A clean, interactive **C#/.NET** console application for managing **Students**, **Instructors**, **Courses**, **Categories**, and **Enrollments** with **Entity Framework Core** (SQL Server LocalDB).
It demonstrates real-world CRUD, **1-N** and **N-N** relationships (via a join entity), and **LINQ-based reports** ‚Äî all through a structured, friendly terminal UI. 

---

## üìù Description

This project is a modular console app that simulates a mini learning platform back office. It covers the full flow:

- Create **Categories** ‚ûú add **Instructors** ‚ûú add **Courses** ‚ûú manage **Students** ‚ûú **Enroll** students ‚ûú generate **Reports** ‚Äî with consistent menus, validation helpers, and a reusable reporting printer. 

---

## ‚ú® Features

- ‚ûï **CRUD** for Students, Instructors, Categories, and Courses  
- üßæ **Enroll students** into courses (prevents duplicate enrollments)  
- üìä **Reports**:  
  - Courses with their instructors  
  - Enrollment count per course  
  - Students with their courses  
  - Course (by ID) with enrolled students  
  - Top 3 courses by enrollments  
- üîé **Query helper**: Get courses under a specific price (service-level)  
- üß± **Separation of concerns**: `Models`, `Services`, `Screens`, `Helpers`, `Data`  
- üß∞ **Generic base service** for common CRUD with EF Core  
- üßØ Graceful console messages + minimal error handling on `SaveChanges()`
